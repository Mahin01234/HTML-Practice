## Topics Covered (by file)

### practice_1.html – Basic HTML
- Document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Headings (`<h1>`–`<h6>`)
- Paragraphs (`<p>`)
- Text formatting: `<b>`, `<i>`, `<u>`, `<strong>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<s>`
- Line breaks (`<br>`) and horizontal rules (`<hr>`)

### practice_2.html – Lists, Links & Images
- Ordered lists (`<ol>`), unordered lists (`<ul>`), definition lists (`<dl>`)
- Anchor tags (`<a>`) with absolute/relative paths, email links, internal links
- Images (`<img>`) with `src`, `alt`, `width`, `height`

### practice_3.html – Semantic & Technical Tags
- `<abbr>`, `<kbd>`, `<code>`, `<pre>`
- `<blockquote>`, `<q>`, `<cite>`

### practice_4.html – Tables & Forms
- Tables: `<table>`, `<tr>`, `<th>`, `<td>`, `colspan`, `rowspan`, `<thead>`, `<tbody>`, `<tfoot>`, `<caption>`
- Forms: `<form>` with `action`, `method`; input types (text, password, email, number, date, radio, checkbox, file, range, color, etc.); `<select>`, `<textarea>`, `<button>`; form validation attributes (`required`, `pattern`, `min`, `max`); `<fieldset>` and `<legend>`

### practice_5.html – Multimedia
- `<audio>` with multiple sources and controls
- `<video>` with controls, poster, and fallback content
- `<iframe>` for embedding external content (YouTube)
- `<figure>` and `<figcaption>`
- `<details>` and `<summary>` for collapsible content
- `<track>` for subtitles (concept)

### practice_6.html – HTML5 APIs
- **Local Storage**: Persistent to‑do list
- **Session Storage**: Counter that survives page reload only during session
- **Geolocation**: Get user location and display on a static map
- **Drag and Drop**: Simple draggable element with drop target
- **Canvas**: Basic drawing (rectangle, circle)
