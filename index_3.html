🌐 Section 1

HTML Web APIs

Web APIs are programming interfaces that extend the functionality of the browser. Examples include:

  • Geolocation API – get user's location
  • Drag and Drop API – make elements draggable
  • Web Storage API – store data locally (localStorage, sessionStorage)
  • Web Workers API – run scripts in background threads
  • Server-Sent Events (SSE) – receive automatic updates from server
